folder src/app
- digunakan untuk routing page, dan memanggil view components

folder src/views
- digunakan untuk menyimpan component berdasarkan pages di src/app
- component ini merupakan component utama html page2 tersebut

folder src/components
- digunakan untuk file component yang kira2 akan digunakan di banyak tempat (global component)

folder src/serviecs
- digunakan untuk menyimpan file berdasarkan resource table
- dimana fungsi di file tersebut adalah yang melakukan network call / komunikasi dengan applikasi Lain


folder src/stores
- digunakan untuk menyimpan auth state management stores

folder src/providers
- digunakan untuk menyimpan component yang akan membungkus children, biasanya memmbungkus seluruh applikasi

folder src/interfaces
- digunakan untuk menyimpan semua interface berdasarkan resource
